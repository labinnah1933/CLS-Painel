## 🛠️ Common issues

### App does not open

- Make sure you downloaded the Windows file from the release page
- If the file is inside a .zip, extract it first
- Try running it again after extraction
- Right-click the file and choose Run as administrator if Windows blocks it

### Windows blocks the file

- Check that the file came from the release page
- Open the file properties and choose Unblock if Windows shows that option
- Try the app again after that

### Results do not load

- Check your internet connection
- Wait a little longer for the scan to finish
- Try a smaller username or one that is spelled right

### The app closes on start

- Restart Windows and try again
- Make sure you use a current Windows version
- Remove and download the latest release again
